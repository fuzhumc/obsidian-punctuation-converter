自动将中文标点转化为对应的英文标点的Obsidian插件.

## 功能特性

- 实时自动转换中文标点为英文标点.
- 可配置转换规则.
- 可随时启用/禁用转换功能.
- 可一键转换当前文档中的所有标点.

## 支持的标点转换

- `，` → `,`
- `。` → `.`
- `：` → `:`
- `；` → `;`
- `！` → `!`
- `？` → `?`
- `【` → `[`
- `】` → `]`
- `（` → `(`
- `）` → `)`
- `《` → `<`
- `》` → `>`
- `"` / `"` → `"`
- `'` / `'` → `'`
- `…` → `...`
- `、` → `\`
- `「` → `{`
- `」` → `}`
- `·` → ` ` `
- `～` → `~`
- `％` → `%`
- `＆` → `&`
- `｜` → `|`

## 使用方法

- 安装插件后,直接输入中文标点即可自动转换.

- 命令面板`切换中文标点转换` - 快速启用/禁用转换功能.

- 命令面板`一键转换当前文档标点` - 转换当前文档中的所有标点.

- 插件设置中可启用/禁用单个标点的转换.
