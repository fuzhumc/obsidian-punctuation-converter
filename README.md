A Obsidian plugin that automatically converts Chinese punctuation to corresponding English punctuation.

## Features

- Automatically convert Chinese punctuation to English punctuation in real-time.
- Configurable conversion rules.
- Enable/disable the conversion feature at any time.
- Convert all punctuation in the current document.

## Supported Punctuation Conversions

- `，` → `,`
- `。` → `.`
- `：` → `:`
- `；` → `;`
- `！` → `!`
- `？` → `?`
- `【` → `[`
- `】` → `]`
- `（` → `(`
- `）` → `)`
- `《` → `<`
- `》` → `>`
- `"` / `"` → `"`
- `'` / `'` → `'`
- `…` → `...`
- `、` → `\`
- `「` → `{`
- `」` → `}`
- `·` → ` ` `
- `～` → `~`
- `％` → `%`
- `＆` → `&`
- `｜` → `|`

## Usage

- After installing the plugin, simply type Chinese punctuation to automatically convert it.

- Use the command palette `切换中文标点转换` to quickly enable/disable the conversion feature.

- Use the command palette `一键转换当前文档标点` to convert all punctuation in the current document.

- Individual punctuation conversions can be enabled/disabled in the plugin settings.
